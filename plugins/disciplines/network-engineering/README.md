# Networking

Network engineering agents providing expertise in protocols, distributed
systems, and service mesh

## Agents

This plugin provides the following specialized agents:

- **Distributed Systems Engineer**
- **Network Protocol Engineer**
- **Service Mesh Engineer**

## Usage

Agents can be invoked using the Agent tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Distributed Systems Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install networking
```
