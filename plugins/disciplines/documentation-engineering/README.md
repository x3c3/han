# Documentation

Agents specialized in technical documentation and knowledge management. Focuses
on documentation architecture and user-focused writing.

## Agents

This plugin provides the following specialized agents:

- **Documentation Engineer**

## Usage

Agents can be invoked using the Agent tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Documentation Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install documentation
```
