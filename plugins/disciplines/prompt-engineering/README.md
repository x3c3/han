# Prompts

Agents specialized in prompt engineering and AI interaction. Focuses on
effective prompt design and AI model optimization.

## Agents

This plugin provides the following specialized agents:

- **Prompt Engineer**

## Usage

Agents can be invoked using the Agent tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Prompt Engineer to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install prompts
```
