# Quality

Agents specialized in quality assurance, testing strategies, and test
architecture. Focuses on ensuring code quality and reliability.

## Agents

This plugin provides the following specialized agents:

- **Quality Strategist**
- **Test Architect**

## Usage

Agents can be invoked using the Agent tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Quality Strategist to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install quality
```
