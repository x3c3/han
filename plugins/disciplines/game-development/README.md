# Games

Game development engineering agents providing expertise in engine architecture,
gameplay systems, and performance optimization

## Agents

This plugin provides the following specialized agents:

- **Game Engine Architect**
- **Game Performance Engineer**
- **Gameplay Engineer**

## Usage

Agents can be invoked using the Agent tool with the appropriate subagent_type or through
natural conversation:

```text
Use the Game Engine Architect to analyze...
```

Each agent brings specialized expertise and perspective
to their domain.

## Installation

```bash
han plugin install games
```
